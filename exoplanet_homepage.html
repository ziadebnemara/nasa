Confusion Matrix Evaluation

This is a Confusion Matrix evaluating a multi-class model's performance on classifying objects as Not Planet, Planet, or Candidate. The diagonal values (912, 623, 1132) represent correct classifications. Off-diagonal values show misclassifications, such as 243 actual 'Not planets' being predicted as 'Candidates.'
